# Syntax
`var[:[type]]? [name] = [value]` Sets a variable to a value.
## Operators
|Syntax|Meaning|
|------|-------|
|`==`|Equality Check|
|`!=`|Inequality Check|
|`\|\|`|OR of values|
|`&&`|AND of values|
|`!`|Opposite of value|
## Types
- Int
- Bool
- Float
- UInt
- Byte
- String
- Object
- Function
